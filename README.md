## 此间所做

古风外壳，现代内核。晓峰不是装饰，而是我面对技术的方式：保持好奇，亲手验证，把复杂系统做成普通人也能使用的工具。

> 不追逐看起来厉害的东西，只做能运行、能验证、能交付的东西。

我的长期方向很清楚：**AI 开发工具、自托管自动化、硬件与科创教育**。
